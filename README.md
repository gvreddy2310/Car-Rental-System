A simple Java Swing (JFrame) based desktop application for managing car rentals.

## Features

* Add new cars
* View available and rented cars
* Rent a car
* Return a car
* GUI built using Java Swing

## Technologies Used

* Java
* Swing / JFrame
* ArrayList Collection

## How to Run

1. Open the project in VS Code or IntelliJ
2. Compile the file:

   ```bash
   javac CarRentalGUI.java
   ```
3. Run the program:

   ```bash
   java CarRentalGUI
   ```
